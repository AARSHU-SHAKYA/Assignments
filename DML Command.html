  • Data Manipulation Language

    • Data manipulation language is a class of SQL statements that are used to query, edit, add and delete row level data from database tables or views. The main DML statements are select, insert, delete and update.

  • Select

    • Select is the most important data manipulation command in structured query language. The SELECT command shows the records of the specified table. It also shows the particular record of a particular column by using the WHERE clause.

    Its code is: SELECT * FROM table_name; (for viewing entire table)

  • Insert

    • INSERT is another most important data manipulation command in structured query language which allows users to imsert data in database tables.

    Its code is: INSERT into table_name(column_Name1, column_Name2,..) Values()

  • Update

    • Update is another most important data manipulation command structured query language, which allows user to update or modify the existing data in database tables.

    Its code is: UPDATE table_name
    SET NAME="" WHERE NAME=""

  • DELETE

    • Delete is a Database manipulation language command which allows Sql users to remove single or multiple existing records from the database tables. This command of data manipulation language does not delete the stored data permanently from the database. We can use the WHERE clause with the delete command to select specific crows from the table.

    Its code is: DELETE FROM table_name WHERE condition;
