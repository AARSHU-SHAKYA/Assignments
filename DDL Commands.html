Day 4

KEYS

  • Data Defination Language
  • Create

    • Used to add a new object to the database.

  • Alter

    • Used to change the structure of the database.

  • Drop

    • Used to remove existing object from the database.

  • Truncate

    • Used to remove all records froma table, including the space allocated to store the data.

  • Rename

    • Used to change the name of the table.

  • Comment

    • Used to add an explanation to the sql code for other team members to review.
